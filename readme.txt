Email newsletter wordpress plugin have option to send HTML mail to registered user also we can send mail to comment posted user. Sending email is much cheaper than most other forms of communication. Email marketing has proven very successful for those who do it right. There is no configuration in this plugin. After activated the plugin you can start send mail to your users.

Sometimes you want an easy way to e-mail all the people who registered, commented on the website, now it's as easy as installing this plug-in.

Send your website update to all your registered/comment posted user using this email newsletter plug-in.
	
Send your website update to all your registered/comment posted user using this email newsletter plugin.

    * Simple.
    * Easy installation.
    * Using this plug-in we can send email to all registered folks.
    * Using this plug-in we can send email to all comment posted folks.
    * Check box option available to select/unselect email from the list.
	* Email subscribe box for front end(available from version 4.0).
	* Integrated the email newsletter plugin & simple contact form plugin

This plug-in broadly consider seven option

1. Plug-in Home	
2. Compose email.	
3. Send mail to registered user.
4. Send mail to comment posted user.	
5. Send mail to subscribed user.
6. Send mail to simple contact form emails.
7. Export emails into CSV file
